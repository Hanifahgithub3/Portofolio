# ☕ Where Logic Meets Latte | Hanifah's Portfolio

Welcome to my personal portfolio repository! This project is a front-end showcase of my journey as an **Informatics Engineering** student who bridges the gap between technical logic and F&B operational excellence.

## 🌟 About This Project
This is a clean, responsive, and interactive personal website built to highlight my professional experience at **Toko Kopi Tuku**, my academic projects in **Cyber Security** and **Data Analysis**, and my technical skill set.

### 🚀 Key Features:
- **Responsive Design:** Optimized for mobile, tablet, and desktop views.
- **Interactive Modals:** View certifications directly with a CSS-based pop-up system.
- **Dynamic Styling:** Custom linear gradients and image overlays for a premium look.
- **Parallax Header:** A smooth scrolling experience with fixed background attachments.

## 🛠️ Built With:
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom layouts using **Flexbox** and **CSS Grid**.
- **Modern UI/UX:** Clean typography and a nature-inspired color palette (#44a424).
